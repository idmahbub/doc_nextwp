### Get `JWT_AUTH_REFRESH_TOKEN`

![Generate refresh token](/generate.png)
JWT tokens are used to secure your WordPress site. The steps to make it are:

- `1` Access to graphql Ide
- `2` Click add new mutation
- `3` Check and fill in the fields: `username, password and refreshToken`
- `4` Switch execute as loged user
- `5` Execute Query
- `6` Copy refreshToken value and paste in env.local file for `JWT_AUTH_REFRESH_TOKEN` value.
