# Install NextWp Worpress Plugin

You can download the Next WP WordPress plugin **[here](/nextwp.zip)**, install and activate it. then, go to the ``settings -> NextWp``, you will see a display like the one below, which indicates that your WordPress site is ready:

![Hello](/nextwpok.png)

If your WordPress is not ready, or an error occurs, you will get a notification like in the image below, you can click on it to get a solution.

![Hello](/nextwper.png)
